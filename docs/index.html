Modern Browser SDK

Browser APIs.
Clean. Typed. SSR-Safe.

@browser/sdk provides minimal, composable wrappers around modern browser APIs like Clipboard, Share, Console, BroadcastChannel, and File System — built for real-world Next.js applications.
